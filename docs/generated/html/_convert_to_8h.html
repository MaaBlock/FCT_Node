Detailed Description

C++ to JavaScript type conversion utilities.

This file provides template functions and specializations for converting C++ types to JavaScript values. It handles various data types including primitives, strings, functions, and lambdas with automatic type detection. Users can extend this system by adding their own template specializations. Users should not include this file directly, but rather include FCT_Node.h instead.
