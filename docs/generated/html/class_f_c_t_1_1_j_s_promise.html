Detailed Description

C++ wrapper for JavaScript Promise objects.

Provides a convenient interface for working with JavaScript Promises from C++ code, including creation, state monitoring, and result retrieval. Supports both creating new promises and wrapping existing ones. Uses std::enable_shared_from_this for safe shared ownership.
