Detailed Description

Convert JavaScript function to C++ std::function.

Template Parameters
ReturnTypeReturn type of the function
ArgsArgument types of the function
Parameters
isolateV8 isolate instance
valueJavaScript function value
Returns
C++ std::function wrapper, nullptr if not a function

Creates a callable C++ function that invokes the JavaScript function. Users can add their own template specializations for custom types by providing template<> specializations of convertFromJS<YourType>.

auto jsFunc = convertFromJS<int, int, int>(isolate, jsFunctionValue);
int result = jsFunc(5, 3); // Calls JavaScript function with arguments
// Custom type specialization example:
template<>
inline MyCustomType convertFromJS<MyCustomType>(NodeEnvironment& env, v8::Local<v8::Value> jsValue) {
// Your custom conversion logic here
return MyCustomType();
}
